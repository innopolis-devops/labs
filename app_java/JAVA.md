# Java

## Spring

```
I used spring and java because it is very powerful framework 
```

### Advantages

```
1) Flexibility
2) Portability
3) Consistent configuration
4) Fast delivery
5) Secure
6) Simpler testing
7) AOP module
```

### Disadvantages

```
1) Fewer guidelines
2) Complexity
3) Hard to learn
4) Uses XML
5) Parallel mechanism
```
