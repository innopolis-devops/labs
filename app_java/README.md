# Devops lab Java application
Grigorii Kostarev

### Overview
This is java application for displaying current Moscow time.

### Build and run
1. Clone repository
2. `cd lab/app_java`
3. Download maven (for example `brew install maven`)
4. Run `mvn spring-boot::run`
5. Check `127.0.0.1:8080`
