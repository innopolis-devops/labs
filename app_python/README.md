# MoscowTimeDisplayer

## :bulb: Overview

MoscowTimeDisplayer is simple web-application, which shows current time in Moscow in the format `HH:MM:SS`

## :rocket: Technological stack

- Flask
- Python


## :hammer: How to set up

install flask: 
`pip install flask`

run the app: 
`py -m flask run`
